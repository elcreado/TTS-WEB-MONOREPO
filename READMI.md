# TTS WEB MONOREPO

Este proyecto está diseñado para leer las interacciones de un chat de Tiktok mediante el uso de un TTS web.
Está dividido en Frontend y Backend.

## Dependencias importantes

- `pnpm`: Gestor de paquetes
- `tiktok-live-connector`: Librería para conectarse al chat de Tiktok
- `ws`: Librería para la comunicación entre el frontend y el backend
- `vite`: Gestor de monorepos

### Estructura del proyecto

- `apps/web`: Aplicación web
- `apps/backend`: Bot de TikTok
- `packages/shared`: Librería compartida
